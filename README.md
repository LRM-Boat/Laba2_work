# Laba2_work


Репозиторий содержит материалы для 2 лабораторной работы. Папки gazebo_lab, image_publisher_zmq и stream_camera_lidar являются пакетами для ROS 2, их необходимо закинуть в папку src рабочего пространства ROS 2. В папке yolo содержится питоновский фаил и весы для работы модели детекции.
